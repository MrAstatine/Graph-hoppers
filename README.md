# Project Setup with uv

This project uses [uv](https://github.com/astral-sh/uv) for Python package management and environment setup.

## Getting Started

1. **Install uv** (if not already installed):
	```sh
	pip install uv
	```

2. **Sync dependencies:**
	```sh
	uv sync
	```
	This will install all dependencies specified in `pyproject.toml`.

3. **Run your project:**
	Use your preferred method (e.g., `python main.py` or as specified in your project).

---
For more details, see the [uv documentation](https://github.com/astral-sh/uv).
